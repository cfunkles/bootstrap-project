Dismissible alerts need a Javascript plug-in. The bootstrap classes needed included: .alert-dissmisible, a button element (with the class of .close), the data-dismiss atribute with value of alert, and an aria-label artiribute with the value of "Close" Notice the C is capitalized

The code for the alert above would look like this

<h2 class="alert alert-warning alert-dissmisible" role="alert>
	This is a Warning alert that is dismissible. Click -->
	<button type="button" class="close" data-dismiss="alert" aria-label="close">
		<span aria-hidden="true"&time;></span>
	</button>
</h2>

Link Alert!
This is a link with the same color as the parent element

Ancor elements with the class .alert-link use the same color as the parent element with the alert class

The code for the alert above would look like this

<div class="alert alert-info" role="alert>
	<a href="#" class="alert-link">
		This is a link with the same color as the parent element
	</a>
</div>

Progress Bars

The Default Progress Bar

30% Complete

To make a default progress bar use this code

<div class="progress">
	<div class="progress-bar" role="progressbar" aria-valuenow=
	"30" aria-valuemin="0" aria-valuemax="100" style="width:30%;">
		<span class="sr-only">30% Complete</span>
	</div>
</div>

Progress Bar with indicator label

If you use the same code in the previous example and then remove the .sr-only class from the span tag, you will see the indication

Combined Progress bars AKA- stacked

To stack progress bars wrap a div with the class .progress around all progress bars and delete the role and aria-value attributes

Maintain a minimum value on the Progress Bar

If you need to show some progress even if the progress is 0% make an inline style of min-width:2em;

0%

	"0" aria-valuemin="0" aria-valuemax="100" style="min-width:2em;">
		<span>0%</span>
	</div>
</div>
